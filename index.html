Strange Estates

Full-stack RESTful site

Utilises Node, Express, MongoDB

An extensive full stack real estate website with listings, users, admins and comments databases. This is a constantly ongoing project, evolving as I learn new skills and add new features.

I enjoy letting my imagination run riot with this site, creating a fictional shady lettings agency specialising in short-term lettings for abandoned structures, for suspiciously high prices...

The front end is built on Bootstrap, with color pallette and styling changes to reflect a darker theme. The landing page contains JavaScript GSAP animation, and features are shown/hidden as appropriate for levels of users.

The back end is built on Node.js, using NPMs such as Express for routing, Passport for user authentification, Mongoose for Mongo integration, Body Parser for passing parameters and Dotenv for hidden variables.

The site contains fully RESTful routes for listings and comments, and partial routing on users and admins.
